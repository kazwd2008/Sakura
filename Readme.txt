R3.kwdは、Rのためのサクラエティタ用設定ファイルです。外部パッケージ未使用のR3.3.1で、次のコマンドにより組み込み関数名を抽出し、さらに予約語を加えたものです。
コマンドは一つずつ実行し、テキストエディタにペーストしてください。

#組み込み関数（外部）
writeClipboard(paste(builtins(), collapse="\r\n"))

#組み込み関数（内部）
writeClipboard(paste(builtins(internal=T), collapse="\r\n"))

使い方は、適当な場所に置き、サクラエディタを開いた後、「共通設定」=> 「強調キーワード」の画面で「セット追加」ボタンを押してセット名「R」を入力、「インポート」ボタンを押して、R3.kwdファイルを指定します。

さらに、対応する括弧を強調するため、サクラエディタで次の設定を行ってください。

　1) 「設定」=> 「タイプ別設定」
  2) 「カラー」タブを選択
  3) 「色指定」一覧から、「対括弧の強調表示」にチェックを入れる

外部パッケージを読み込んだ後、組み込み関数名を抽出すれば、パッケージの関数にも対応すると思います。ご自由に、よりよい設定ファイルに改良いただければ幸いです。
